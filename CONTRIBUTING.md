# Contributing to Raeen Dashboard

First off, thank you for considering contributing to Raeen Dashboard! It's people like you that make this tool better for everyone.

## Code of Conduct

By participating in this project, you are expected to uphold our Code of Conduct:
- Be respectful and inclusive.
- Focus on constructive feedback.
- Help others where possible.

## How Can I Contribute?

### Reporting Bugs
- Check the [Issues](https://github.com/YOUR_USERNAME/NewTab_Lite/issues) page to see if the bug has already been reported.
- If not, open a new issue with a clear title and description.
- Include steps to reproduce the bug and details about your environment (Browser version, OS).

### Suggesting Enhancements
- Open a new issue with the tag `enhancement`.
- Describe the feature you'd like to see and why it would be useful.

### Pull Requests
- Fill in the pull request template (if provided).
- Ensure your code follows the existing style of the project.
- Test your changes thoroughly.

## Development Setup

1. Fork the repo.
2. Load the extension unpacked in your browser.
3. Make your changes.
4. Refresh the extension to test.

## Style Guide
- Use consistent indentation (2 spaces).
- Comment your code where logic is complex.
- Use semantic HTML tags.
